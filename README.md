### Generate Grammar

To get the lexer and parser generated from the grammar(s), We can simply run:

####  Linux/Mac
```bash
./gradlew generateGrammarSource
```
#### Windows
```bash
gradlew generateGrammarSource
```
Then we can also run:
#### Linux/Mac
```bash
./gradlew idea
```
#### Windows
```bash
gradlew idea
```
